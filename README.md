# 📊 Exploratory Data Analysis (EDA) & Data Preprocessing

A collection of hands-on Exploratory Data Analysis (EDA) and Data Preprocessing projects using real-world datasets. This repository focuses on understanding data, discovering patterns, cleaning datasets, engineering features, and preparing data for Machine Learning workflows.

---

## 🎯 Objectives

- Understand dataset structure and characteristics
- Perform statistical and visual analysis
- Handle missing values and duplicates
- Detect and treat outliers
- Apply feature engineering techniques
- Encode categorical variables
- Prepare data for Machine Learning models
