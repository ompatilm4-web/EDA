# 📊 Exploratory Data Analysis & Data Preprocessing

> A collection of hands-on EDA and Data Preprocessing projects using real-world datasets — focused on understanding data, discovering patterns, cleaning datasets, engineering features, and preparing data for Machine Learning workflows.

---

## 🎯 Objectives

| Goal | Description |
|------|-------------|
| 🔍 Understand | Grasp dataset structure and characteristics |
| 📈 Analyse | Perform statistical and visual analysis |
| 🧹 Clean | Handle missing values, duplicates, and inconsistencies |
| 🚨 Detect | Identify and treat outliers |
| ⚙️ Engineer | Apply feature engineering techniques |
| 🔠 Encode | Transform categorical variables |
| 🤖 Prepare | Ready data for Machine Learning models |

---

## 🛠️ Topics Covered

### 🧹 Data Cleaning
- Missing Value Analysis
- Duplicate Detection & Removal
- Data Type Conversion
- Inconsistent Data Handling

### ⚙️ Feature Engineering
- Feature Creation & Transformation
- Feature Combination
- Domain-Based Feature Construction

### 🔠 Encoding Techniques
- Label Encoding · One-Hot Encoding
- Ordinal Encoding · Target Encoding

### 🔎 Exploratory Data Analysis
- Univariate, Bivariate & Multivariate Analysis
- Correlation & Distribution Analysis

### 📊 Data Visualization
- Histograms · Count Plots · Box Plots
- Scatter Plots · Pair Plots · Heatmaps

### 🗂️ Data Preparation
- Scaling & Normalization
- Handling Imbalanced Data
- Train-Test Split Preparation
